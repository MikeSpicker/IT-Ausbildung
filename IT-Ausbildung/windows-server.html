Windows Server Grundlagen

Windows Server ist ein Betriebssystem von Microsoft, das speziell für den Einsatz in Unternehmen entwickelt wurde. Es dient zur Verwaltung von Netzwerken, Benutzern, Serverdiensten und Anwendungen.

Warum Windows Server?

  • Zentrale Benutzer- und Ressourcenverwaltung
  • Skalierbarkeit und Sicherheit
  • Virtualisierungsmöglichkeiten mit Hyper-V
  • Unterstützung für Cloud- und On-Premise-Infrastrukturen

Systemanforderungen

Prozessor

64-Bit, mindestens 1,4 GHz (empfohlen: 2 GHz oder mehr)

RAM

Mindestens 2 GB (empfohlen: 8 GB oder mehr)

Festplatte

Mindestens 32 GB freier Speicherplatz

Netzwerk

Gigabit-Ethernet empfohlen

Windows Server Rollen

Rolle Funktion PowerShell-Befehl
Active Directory (AD DS) Verwaltung von Benutzern und Gruppen Install-WindowsFeature -Name AD-Domain-Services
DNS-Server Namensauflösung von Netzwerkgeräten Install-WindowsFeature -Name DNS
DHCP-Server Automatische IP-Adressvergabe Install-WindowsFeature -Name DHCP
Datei- und Druckserver Zentrale Datei- und Druckverwaltung Install-WindowsFeature -Name File-Services
Hyper-V Virtualisierung von Gäste-Betriebssystemen Install-WindowsFeature -Name Hyper-V
Webserver (IIS) Hosten von Websites und Webanwendungen Install-WindowsFeature -Name Web-Server

Grundlagen des Windows Servers

Was ist Windows Server?

Windows Server ist die Server-Edition von Windows und bietet folgende Funktionen:

  • Active Directory (AD) – Verwaltung von Benutzern, Gruppen und Rechten
  • Datei- und Druckserver – Gemeinsame Nutzung von Dateien und Druckern
  • DNS- & DHCP-Server – Namensauflösung und IP-Adressenvergabe
  • Hyper-V (Virtualisierung) – Virtuelle Maschinen erstellen
  • Rollen & Features – Anpassbare Server-Funktionalität

Aktuelle Versionen:

  • Windows Server 2016
  • Windows Server 2019
  • Windows Server 2022 (aktuellste Version)

Wichtige Funktionen

  • Active Directory:

    Zentrale Verwaltung von Benutzern, Gruppen und Ressourcen

  • DNS-Server:

    Namensauflösung im Netzwerk

  • DHCP-Server:

    Automatische IP-Adressvergabe

  • Datei- und Druckserver:

    Zentrale Speicherung und Druckverwaltung

Active Directory (AD) Grundlagen

Active Directory (AD) ist ein Verzeichnisdienst zur Verwaltung von Benutzern, Gruppen und Computern im Netzwerk.

Wichtige AD-Konzepte:

  • Domäne – Netzwerk mit zentraler Benutzerverwaltung
  • Organizational Units (OU) – Struktur für Benutzer & Gruppen
  • Gruppenrichtlinien (GPOs) – Regeln für Benutzer & Computer
  • Benutzerverwaltung – Erstellen, Ändern & Löschen von Konten

Beispiel: Benutzer erstellen (PowerShell)

New-ADUser -Name "Max Mustermann" -GivenName "Max" -Surname "Mustermann" -SamAccountName "m.mustermann" -UserPrincipalName "m.mustermann@firma.local" -Path "OU=Mitarbeiter,DC=firma,DC=local" -AccountPassword (ConvertTo-SecureString "Passwort123" -AsPlainText -Force) -Enabled $true

Praktische Übungen

Übung 1: Active Directory

Erstellen Sie eine neue Benutzergruppe und fügen Sie Benutzer hinzu.

New-ADGroup -Name "IT-Abteilung" -GroupScope Global

Übung 2: Sicherheit

Konfigurieren Sie die Windows Firewall für einen Webserver.

New-NetFirewallRule -DisplayName "Web Server" -Direction Inbound -Protocol TCP -LocalPort 80,443 -Action Allow

Übung 3: Speicherverwaltung

Erstellen Sie einen neuen Datenträger und formatieren Sie ihn.

Initialize-Disk -Number 1 -PartitionStyle GPT
