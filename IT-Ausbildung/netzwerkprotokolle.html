TCP vs. UDP

TCP (Transmission Control Protocol)

  • Verbindungsorientiert
  • Garantierte Zustellung
  • Reihenfolge wird eingehalten
  • Flusskontrolle
  • Beispiele: HTTP, FTP, SMTP

UDP (User Datagram Protocol)

  • Verbindungslos
  • Keine Zustellungsgarantie
  • Keine Reihenfolgegarantie
  • Schneller als TCP
  • Beispiele: DNS, DHCP, Streaming

Übungsaufgaben

Ordnen Sie die folgenden Anwendungen dem passenden Transportprotokoll (TCP oder UDP) zu:

  • E-Mail-Versand
  • Online-Gaming
  • Webseiten-Aufruf
  • VoIP-Telefonie
  • E-Mail-Versand: TCP (Zuverlässige Übertragung wichtig)
  • Online-Gaming: UDP (Geschwindigkeit wichtiger als Zuverlässigkeit)
  • Webseiten-Aufruf: TCP (Vollständige und korrekte Übertragung wichtig)
  • VoIP-Telefonie: UDP (Echtzeitübertragung wichtiger als Vollständigkeit)
