Das OSI-Modell

Was ist das OSI-Modell?

Das OSI-Modell (Open Systems Interconnection Model) ist ein Referenzmodell für Netzwerkprotokolle. Es besteht aus sieben Schichten, die die Kommunikation zwischen verschiedenen Systemen standardisieren.

Schicht Name Funktion Beispiele
7 Anwendungsschicht Schnittstelle zu Anwendungen HTTP, FTP, SMTP
6 Darstellungsschicht Datendarstellung und -verschlüsselung SSL/TLS, JPEG, ASCII
5 Sitzungsschicht Sitzungsverwaltung NetBIOS, RPC
4 Transportschicht Ende-zu-Ende-Verbindung TCP, UDP
3 Vermittlungsschicht Routing und Adressierung IP, ICMP
2 Sicherungsschicht Fehlererkennung und -korrektur Ethernet, WLAN
1 Bitübertragungsschicht Physische Übertragung Kabel, Funk

Datenfluss im OSI-Modell

Sender

  • 1. Daten werden in der Anwendungsschicht erstellt
  • 2. Jede Schicht fügt ihre eigenen Header hinzu
  • 3. Daten werden nach unten weitergereicht
  • 4. Physische Übertragung in Schicht 1

Empfänger

  • 1. Empfang der Daten in Schicht 1
  • 2. Daten werden nach oben weitergereicht
  • 3. Jede Schicht entfernt ihren Header
  • 4. Anwendung erhält ursprüngliche Daten

Übungsaufgaben

Ordnen Sie die folgenden Protokolle den richtigen OSI-Schichten zu:

  • HTTP
  • TCP
  • IP
  • Ethernet
  • HTTP - Schicht 7 (Anwendungsschicht)
  • TCP - Schicht 4 (Transportschicht)
  • IP - Schicht 3 (Vermittlungsschicht)
  • Ethernet - Schicht 2 (Sicherungsschicht)
