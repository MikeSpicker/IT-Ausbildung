SQL Basics

Was ist SQL?

SQL (Structured Query Language) ist die Standardsprache für:

  • Datenbankabfragen
  • Datenmanipulation
  • Datenbankadministration
  • Datendefinition

Wichtige Konzepte

  • Tabellen und Relationen
  • Primär- und Fremdschlüssel
  • CRUD-Operationen
  • Joins und Verknüpfungen

Wichtige SQL-Befehle

Daten Abfragen

SELECT

Daten aus einer Tabelle auswählen

WHERE

Bedingungen für die Auswahl festlegen

JOIN

Tabellen miteinander verknüpfen

GROUP BY

Daten gruppieren

Daten Manipulieren

INSERT

Neue Daten einfügen

UPDATE

Bestehende Daten aktualisieren

DELETE

Daten löschen

MERGE

Daten zusammenführen

Datenbank-Struktur

CREATE

Neue Tabellen erstellen

ALTER

Tabellenstruktur ändern

DROP

Tabellen löschen

INDEX

Indizes verwalten

Best Practices

Performance

  • ✅ Indizes richtig einsetzen
  • ✅ Spezifische Spalten statt *
  • ✅ WHERE vor JOIN
  • ❌ Unterabfragen vermeiden

Sicherheit

  • ✅ Prepared Statements
  • ✅ Input validieren
  • ✅ Minimale Rechte
  • ❌ SQL-Injection vermeiden

Wartbarkeit

  • ✅ Klare Namensgebung
  • ✅ Kommentare nutzen
  • ✅ Formatierung
  • ❌ Redundanz vermeiden
