Dateisysteme

Grundlagen der Dateisysteme

Ein Dateisystem organisiert die Speicherung und Verwaltung von Daten auf Speichermedien.

Grundlegende Konzepte

  • Datei

    Eine benannte Sammlung von Daten

  • Verzeichnis

    Container für Dateien und andere Verzeichnisse

  • Pfad

    Eindeutige Adresse einer Datei im System

Dateieigenschaften

  • Name und Erweiterung
  • Größe und Speicherort
  • Zugriffsrechte
  • Zeitstempel (Erstellung, Änderung)
  • Attribute (versteckt, schreibgeschützt)

Dateisystemtypen

Dateisystem Betriebssystem Max. Dateigröße Besonderheiten
NTFS Windows 16 TB - 256 TB Journaling, Verschlüsselung, Quotas
ext4 Linux 16 TB - 64 PB Journaling, Extents, schnelle FSck
APFS macOS 8 EiB Snapshots, Verschlüsselung, SSD-optimiert
FAT32 Universal 4 GB Breite Kompatibilität

Dateisystemoperationen

Grundoperationen

  • Erstellen

    Neue Dateien und Verzeichnisse anlegen

  • Lesen/Schreiben

    Zugriff auf Dateiinhalte

  • Löschen

    Entfernen von Dateien und Verzeichnissen

Erweiterte Operationen

  • Defragmentierung

    Optimierung der Dateianordnung

  • Backup

    Sicherung von Dateien und Verzeichnissen

  • Komprimierung

    Platzsparende Speicherung

Übungsaufgaben

Welches Dateisystem würden Sie in folgenden Szenarien wählen?

  • USB-Stick für Datenaustausch zwischen verschiedenen Betriebssystemen
  • Systempartition eines Windows 10 Rechners
  • Netzwerkspeicher für große Videodateien
Musterlösung:
  • USB-Stick: FAT32 (beste Kompatibilität)
  • Windows 10: NTFS (Sicherheit und Performance)
  • Videodateien: NTFS oder ext4 (große Dateien)
