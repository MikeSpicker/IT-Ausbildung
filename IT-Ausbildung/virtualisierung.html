Was ist Virtualisierung?

Virtualisierung ist eine Technologie, die es ermöglicht, physische IT-Ressourcen in virtuelle Umgebungen umzuwandeln.

Hauptvorteile:
  • Bessere Ressourcennutzung
  • Kostenersparnis
  • Einfache Verwaltung
  • Erhöhte Sicherheit
  • Portabilität

Arten der Virtualisierung

  • Hyper-V (Microsoft)
  • VMware vSphere
  • Proxmox VE
  • KVM
  • Citrix Hypervisor (ehemals XenServer)
  • Oracle VM Server
  • Red Hat Virtualization
  • OpenVZ

  • Docker
  • Kubernetes
  • LXC (Linux Containers)
  • Podman
