Was ist Linux?

Linux ist ein kostenloses, offenes Betriebssystem, das auf dem Unix-System basiert. Es ist besonders beliebt bei Entwicklern und für Server-Systeme.

1. Dateisystem

Linux verwendet eine baumartige Struktur für Dateien und Ordner:

  • / - Das Wurzelverzeichnis
  • /home - Benutzerverzeichnisse
  • /etc - Systemkonfigurationsdateien
  • /bin - Wichtige Systemprogramme

2. Benutzerrechte

Linux hat ein ausgeklügeltes Berechtigungssystem:

  • Lesen (r)
  • Schreiben (w)
  • Ausführen (x)

Diese Rechte können für den Besitzer, die Gruppe und andere Benutzer gesetzt werden.

3. Wichtige Befehle

Hier sind die wichtigsten Befehle für den Anfang:

  • ls - Zeigt Dateien und Ordner an
  • cd - Wechselt das Verzeichnis
  • pwd - Zeigt das aktuelle Verzeichnis
  • mkdir - Erstellt einen neuen Ordner
  • rm - Löscht Dateien

4. Terminal

Das Terminal ist die Kommandozeile von Linux. Hier können Sie:

  • Befehle direkt eingeben
  • Mit dem System interagieren
  • Programme ausführen
  • Dateien verwalten

5. Paketverwaltung

Linux verwendet Paketmanager zum Installieren von Software:

  • apt (Ubuntu/Debian)
  • yum (Red Hat/Fedora)
  • pacman (Arch Linux)

Beispiel: sudo apt install programmname

Praktische Übungen

Vertiefen Sie Ihre Linux-Kenntnisse mit praktischen Übungen

Übung 1: Navigation

Lernen Sie, sich im Linux-Dateisystem zu bewegen:

$ pwd
/home/user
$ cd /home
$ mkdir linux_uebungen
$ cd linux_uebungen
  1. Öffnen Sie das Terminal
  2. Geben Sie pwd ein
  3. Navigieren Sie mit cd /home
  4. Erstellen Sie einen Ordner
  5. Wechseln Sie in den Ordner
Tipp: Nutzen Sie die Tab-Taste zur Autovervollständigung

Übung 2: Dateiverwaltung

Üben Sie den Umgang mit Dateien:

$ touch test.txt
$ nano test.txt
$ cat test.txt
Hallo Linux!
  1. Erstellen Sie eine Datei
  2. Öffnen Sie den Editor
  3. Schreiben Sie Text
  4. Speichern (Strg+O)
  5. Beenden (Strg+X)

Übung 3: Berechtigungen

Lernen Sie das Berechtigungssystem kennen:

$ ls -l
-rw-r--r-- 1 user user 11 Mar 19 10:00 test.txt
$ chmod 755 test.txt
$ ls -l
-rwxr-xr-x 1 user user 11 Mar 19 10:00 test.txt
Berechtigungen verstehen
Lesen (r) 4
Schreiben (w) 2
Ausführen (x) 1

Übung 4: Paketverwaltung

Installieren Sie ein Programm:

  1. Aktualisieren Sie die Paketliste: sudo apt update
  2. Installieren Sie ein Programm (z.B. htop): sudo apt install htop
  3. Starten Sie das Programm: htop

Übung 5: Systeminformationen anzeigen

Lernen Sie wichtige Systembefehle:

  1. Systeminformationen anzeigen: uname -a
  2. Speichernutzung prüfen: free -h
  3. Festplattennutzung anzeigen: df -h
Lernziele

Nach Abschluss dieser Übungen sollten Sie:

  • Sich im Dateisystem zurechtfinden können
  • Dateien erstellen und verwalten können
  • Berechtigungen verstehen und anpassen können
  • Programme installieren können
  • Wichtige Systeminformationen abrufen können
