Prozessoren und Speicher

Grundlagen der Computer-Hardware und Speichertechnologien

CPU (Central Processing Unit)

Die CPU ist das "Gehirn" des Computers und führt die grundlegenden arithmetischen, logischen und Ein-/Ausgabe-Operationen aus.

Wichtige CPU-Komponenten

  • ALU (Arithmetic Logic Unit)

    Führt mathematische und logische Operationen aus

  • Steuerwerk

    Steuert den Programmablauf und koordiniert alle Komponenten

  • Register

    Schnelle Zwischenspeicher für Daten und Befehle

  • Cache

    Schneller Zwischenspeicher für häufig benötigte Daten

Wichtige CPU-Kennzahlen

Taktfrequenz

Typisch: 3.5 - 5.0 GHz

Kerne

Typisch: 6 - 16 Kerne

Cache-Größe

L3: bis zu 64MB

Speicherhierarchie

Speichertyp Geschwindigkeit Kapazität Kosten pro GB
CPU Register < 1ns Bytes Sehr hoch
Cache (L1-L3) 2-20ns KB-MB Hoch
RAM 50-100ns GB Mittel
SSD 100µs TB Niedrig
HDD 10ms TB Sehr niedrig

RAM (Random Access Memory)

RAM-Typen

  • DDR4

    Aktueller Standard, 2133-3200 MHz

  • DDR5

    Neuer Standard, 4800-6400 MHz

Wichtige Eigenschaften

  • Flüchtiger Speicher
  • Direkter Zugriff
  • Hohe Geschwindigkeit
  • Arbeitspeicher für aktive Programme

Übungsaufgaben

Ordnen Sie die folgenden Speichertypen nach Geschwindigkeit (von schnell nach langsam):

  • RAM
  • HDD
  • CPU Cache
  • SSD
  1. CPU Cache (schnellster)
  2. RAM
  3. SSD
  4. HDD (langsamster)
