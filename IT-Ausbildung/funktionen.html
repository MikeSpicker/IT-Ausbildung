Funktionen in der Programmierung

Was sind Funktionen?

Funktionen sind wiederverwendbare Codeblöcke, die eine bestimmte Aufgabe ausführen. Sie helfen uns dabei, Code zu organisieren und Wiederholungen zu vermeiden.


def begruessung(name):
    return f"Hallo {name}!"

# Funktion aufrufen
ergebnis = begruessung("Max")
print(ergebnis)  # Ausgabe: Hallo Max!

Funktionsparameter

  • Pflichtparameter
  • Optionale Parameter
  • Keyword Arguments

def berechne_summe(a, b=0, *args):
    summe = a + b
    for zahl in args:
        summe += zahl
    return summe

Übungsaufgaben

Erstellen Sie eine Funktion, die zwei Zahlen multipliziert und das Ergebnis zurückgibt.


def multipliziere(a, b):
    return a * b

# Test
ergebnis = multipliziere(4, 5)
print(ergebnis)  # Ausgabe: 20
