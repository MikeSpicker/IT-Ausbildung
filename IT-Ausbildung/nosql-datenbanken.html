NoSQL Datenbanktypen

Dokumentenorientiert

Speichert Daten in flexiblen Dokumenten (JSON/BSON)

  • ✓ MongoDB
  • ✓ CouchDB
  • ✓ RavenDB

Key-Value

Speichert Datenpaare als Schlüssel-Wert

  • ✓ Redis
  • ✓ DynamoDB
  • ✓ Riak

Wide-Column

Speichert Daten in flexiblen Spalten

  • ✓ Cassandra
  • ✓ HBase
  • ✓ ScyllaDB

Graph

Speichert Daten als Knoten und Beziehungen

  • ✓ Neo4j
  • ✓ ArangoDB
  • ✓ OrientDB
